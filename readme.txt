== Description ==

This plugin provides a widget that let you select the comments you want to display as featured or important.

Go to the Widgets page and you will find a new widget named Featued Comments. Just active it and enjoy!

== Installation ==

1. Upload `featured-comments-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the Widgets page under the Appearance menu and active the Featured Comments widget
4. Customize the widget options and select the comments you want to feature
